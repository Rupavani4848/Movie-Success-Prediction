The entertainment industry involves massive investments, making movie success prediction crucial.
This project builds a model to classify movies as successful or not using Naive Bayes, Logistic Regression, and SVM.
Datasets include features like budget, genre, cast, director, release date, and past box office trends, with preprocessing and feature selection applied.
The findings highlight how machine learning can provide valuable insights for producers, investors, and marketers in forecasting movie performance.
